# Data-Warehouse-SQL-Server-Project
## Objective
Build a data warehouse using SQL Server to prepare sales data for analytics. 

## Specifications
- **Data Sources:** Import data from ERP and CRM as CSV files.
- **Data Quality:** Clean and resolve data quality issues.
- **Data Quality:** Combine both data sources into a single, unified data model.
- **Scope:** Only use the most recent dataset. No historical tracking.
- **Documentation:** Provide clear documentation of the data model to support non-technical teams.
